# nostr-mcp spec

Canonical specification for the `nostr-mcp` tool naming and registry.

This repo defines:
- the canonical tool naming scheme
- the tool registry (names, status, NIP mapping)
- the spec version and changelog

Scope: standardized tool names and a shared registry for MCP tool servers that operate on Nostr data.

Files:
- SPEC.md
- registry/tools.json
- CHANGELOG.md
