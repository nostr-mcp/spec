# nostr-mcp spec

Canonical specification for the `nostr-mcp` tool naming and schemas.

This repo defines:
- the canonical tool naming scheme
- the tool registry (names, status, NIP mapping, schemas)
- error semantics and versioning policy
- a roadmap for introducing planned tools

Scope: standardized tool names and schemas for MCP tool servers that operate on Nostr data.
